# NirvanaRemap

Esta solução contém diferentes interfaces de usuário para o mecanismo de remapeamento de controles. A implementação baseada em MAUI permanece no projeto **NirvanaMAUI**, enquanto a nova interface multiplataforma utiliza **AvaloniaUI** no projeto **NirvanaAvalonia**.

O projeto Avalonia compartilha os mesmos serviços e modelos do backend, oferecendo experiência similar ao aplicativo MAUI.
